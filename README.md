# shopApp

以下内容将协助你运行此使用vite搭建的vue3项目

## IDE 推荐

[VSCode](https://code.visualstudio.com/) + [Volar](https://marketplace.visualstudio.com/items?itemName=Vue.volar) .

## 启动项目
请先在项目的根目录下打开git bash(index.html所处位置),
依次输入以下命令:
### 安装依赖项
```sh
npm install
```

### 启动了本地开发服务器
```sh
npm run dev
```

### 打包项目

```sh
npm run build
```
